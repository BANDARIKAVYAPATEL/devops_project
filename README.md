🚨 Disaster Response System

A full-stack MERN application designed to help people report disasters and enable rescue teams to respond quickly and efficiently.

🌍 Overview

📌 This system allows users to report emergencies such as floods, fires, and accidents.
📌 Reports are instantly shared with rescue teams and admins.
📌 Helps improve communication and reduce response time.

🎯 Objectives

✔️ Easy disaster reporting
✔️ Faster rescue response
✔️ Centralized report management
✔️ Better emergency coordination

👥 User Roles
👤 User

🔹 Register & login
🔹 Report disasters
🔹 Upload images 📷
🔹 Share location 📍
🔹 View safety tips

🚑 Rescue Team

🔹 View reports
🔹 Update status
🔹 Mark tasks as completed ✅
🔹 Handle alerts 🚨

🛠 Admin

🔹 View all reports 📊
🔹 Manage users
🔹 Send alerts 📢
🔹 Monitor system

⚙️ How to Run
# Clone repository
git clone https://github.com/BANDARIKAVYAPATEL/devops_project.git

# Backend setup
cd backend
npm install
npm start

# Frontend setup
cd frontend
npm install
npm start
